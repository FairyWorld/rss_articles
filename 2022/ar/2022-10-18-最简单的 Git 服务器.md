# 最简单的 Git 服务器

## 服务器操作

使用一个新用户，后期的灵活性会大一点。

```bash
$ sudo mkdir /home/git
$ sudo useradd git
$ sudo mkdir -m 700 /home/git/.ssh
$ sudo cp ~/.ssh/authorized_keys /home/git/.ssh/
```

## 本机操作

```bash
$ ssh pn41-git git init --bare reading-list.git
$ cd reading-list
$ git remote add pn41 pn41-git:reading-list.git
$ git push pn41 master
```
